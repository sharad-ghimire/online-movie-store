# Online Movie Store (OMS)
A web site and web services for Online Movie Store(OMS) that allows users to look-up and purchase available movies. 

# Major Technology Stacks
- Java 
- XML 
- XSLT and XPATH
- Bootstrap 
- JAXB, JAX-RS and JAX-WS
